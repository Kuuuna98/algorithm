## 커밋 규칙

```
{Keyword}: [#{Issue}] {Site} {난이도} {제목}
```

- 레포지토리 내 Issues에 문제와 관련한 자료구조/알고리즘 이슈 추가
- 문제 Commit 시 관련 이슈 넘버 태그

#### Keyword

- `solve`: 새로운 문제에 대한 풀이
- `refactor`: 등록한 풀이에 대한 코드 리팩토링
- `docs`: readme 수정

#### Site

- `bj`: 백준
- `ct`: 코드트리
- `pg`: 프로그래머스
